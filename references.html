References¶

Citation¶

To cite PDAL in publications use:

PDAL Contributors, 2018. PDAL Point Data Abstraction Library. doi:10.5281/zenodo.2556738

A BibTeX entry for LaTeX users is

@misc{pdal_contributors_2018_2556738,

author = {PDAL Contributors}, title = {PDAL Point Data Abstraction Library}, month = nov, year = 2018, doi = {10.5281/zenodo.2556738}, url = {https://doi.org/10.5281/zenodo.2556738}

}
