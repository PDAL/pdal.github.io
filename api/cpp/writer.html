boost::uint64_t write(boost::uint64_t targetNumPointsToWrite = 0, boost::uint64_t startingPosition = 0, boost::uint64_t chunkSize = 0)

Read the given number of points (or less, if the reader runs out first), and then write them out to wherever.

Returns total number of points actually written.

Return
The number of points that were written.
Parameters
  • targetNumPointsToWrite -

    The number of points to write. If given number of points is 0, do as many points as the reader supplies to us.

  • startingPosition -

    The starting position to start reading from This position is arrived at by the prevStage’s seekImpl.

  • chunkSize -

    The chunk size, or size of the internal Writer‘s buffer, to use. If no chunkSize is specified, a chunkSize that defaults to the Stage::getNumPoints() is used.
