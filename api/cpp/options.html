class pdal::Options

An Options object is just a map of names to Option objects.

Dumped as XML, an Options object with two Option objects looks like this:

<?xml...>
<Option>
  <Name>myname</name>
  <Value>17</value>
  <Description>my descr</description>
</Option>
<Option>
  <Name>myname2</name>
  <Value>13</value>
  <Description>my descr2</description>
</Option>
