class pdal::Log

pdal::Log is a logging object that is provided by pdal::Stage to facilitate logging operations.

Constructors
Log(std::string const & leaderString, std::string const & outputName)

Constructs a pdal::Log instance.

Parameters
  • leaderString -

    A string to presage all log entries with

  • outputName -

    A filename or one of ‘stdout’, ‘stdlog’, or ‘stderr’ to use for outputting log information.

Log(std::string const & leaderString, std::ostream * v)

Constructs a pdal::Log instance.

Parameters
  • leaderString -

    A string to presage all log entries with

  • v -

    An existing std::ostream to use for logging (instead of the the instance creating its own)

Destructor
~Log()

The destructor will clean up its own internal log stream, but it will not touch one that is given via the constructor.
