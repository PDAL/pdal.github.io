class pdal::Schema

A pdal::Schema is a composition of pdal::Dimension instances that form a point cloud.

bool setDimension(Dimension const & dim)

overwrites an existing Dimension with the same name as dim

Parameters
  • dim -

    the Dimension instance that contains the name and namespace to overwrite in the Schema.

    Note

    If no namespace is given, the first dimension with a matching pdal::Dimension::getName() will be overwritten. To be sure, have set the namespace of the pdal::Dimension using pdal::Dimension::setNamespace() beforehand.
