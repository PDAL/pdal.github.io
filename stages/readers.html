ReadersÂ¶

Readers are data providers to Pipeline operations. A reader might provide a simple file type, like readers.text, a complex database like readers.oci, or a network service like readers.greyhound.

Note

Readers provide Dimensions to Pipeline. PDAL attempts to normalize common dimension types, like X, Y, Z, or Intensity, which are often found in LiDAR point clouds. Not all dimension types need to be fixed, however. Database drivers typically return unstructured lists of dimensions.
